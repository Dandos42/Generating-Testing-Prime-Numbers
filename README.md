# BPC-AKR Python Primitive Generation

## Description
This project focuses on the problem of generating large prime numbers and subsequently testing these, as well as other large prime numbers, using mathematical operations, specifically by employing at least two different algorithms. It includes a theoretical introduction to prime numbers, their main properties, the problem associated with inverse factorization, and their importance in modern cryptographic systems. The subsection on prime number generation specifically focuses on the algorithms that can be used for generation and that were ultimately employed. Linked to this is a subsection on primality testing, specifically the types of primality tests and their descriptions. The project also incorporates the ability to list all prime numbers up to a number specified by the user for the purpose of identifying prime numbers. It utilizes file operations, specifically the use of data from a file as input for primality testing, or the export of prime numbers to a separate file for potential future use. Finally, the application is capable of logging ongoing events, including the duration of prime number generation and testing. The last and most comprehensive chapter contains the study of the project developed so far in the Python programming language and the next steps necessary for the project’s successful completion.
